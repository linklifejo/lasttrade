# 🔥 불타기 vs 💧 물타기 전략 완벽 가이드

---

## 📋 **전략 개요**

### **불타기 (FIRE) 전략**
```
승자에 베팅 = 수익 나는 종목에 추가 매수
```

**핵심 원리:**
```
1차 매수 (10%) → 테스트
  ↓
+3% 달성 → 2차 매수 (90%) → 확신
  ↓
TS 활성화 (+1.5%) → 수익 극대화
  ↓
고점 추적 → 최적 익절
```

---

### **물타기 (WATER) 전략**
```
평단가 낮추기 = 손실 나는 종목에 추가 매수
```

**핵심 원리:**
```
1차 매수 (10%) → 진입
  ↓
-1% 하락 → 2차 매수 (10%)
  ↓
-1% 하락 → 3차 매수 (10%)
  ↓
... (10회까지 분할)
  ↓
평단가 낮춤 → 반등 시 익절
```

---

## 🎯 **프리셋 설정 (자동 적용)**

### **🔥 몰빵 불타기**
```python
target_stock_count = 1        # 1개 종목 집중
split_buy_cnt = 2             # 2회 분할
initial_buy_ratio = 10%       # 초기 10% 매수
single_stock_strategy = FIRE  # 불타기
single_stock_rate = 3.0%      # +3% 이상 시 추가 매수
stop_loss_rate = -5.0%        # 빠른 손절
take_profit_rate = 10.0%      # 목표 익절
trailing_stop = 활성화         # TS로 수익 극대화
  - 활성화: +1.5%
  - 콜백: -0.5%
time_cut = 30분 / 1.0%        # 기회비용 최소화
```

**시나리오:**
```
삼성전자 진입 (1,000만원)
  ↓
1차 매수: 100만원 (10%)
  ↓
+3% 달성 → 2차 매수: 900만원 (90%)
  ↓
+1.5% 도달 → TS 활성화
  ↓
고점 +5.8% → -0.5% 하락 → 익절 +5.3%
  ↓
수익: +53만원
```

## 📊 **전략별 비교**

| 항목 | 불타기 (FIRE) | 물타기 (WATER) |
|------|---------------|----------------|
| **철학** | 승자에 베팅 | 평단가 낮추기 |
| **분할 횟수** | 2회 (빠른 집중) | 10회 (점진적) |
| **초기 매수** | 10% (테스트) | 10% (시작) |
| **추가 매수** | +3% 이상 | -1% 하락 |
| **손절** | -5% (빠름) | -10% (보호) |
| **익절** | TS +1.5% | TS +2.0% |
| **리스크** | 낮음 | 높음 |
| **승률** | 높음 | 중간 |
| **수익률** | 높음 | 중간 |
| **추천도** | ✅✅ 강력 추천 | ⚠️ 고급 전략 |

## 🔥 **불타기 전략 상세**

### **매수 로직:**
```
1차 매수 (10%):
  ├─ 조건: 신규 진입
  ├─ 금액: 종목당 할당액 × 10%
  └─ 목적: 테스트 (틀리면 빠른 손절)

2차 매수 (90%):
  ├─ 조건: +3% 이상 달성
  ├─ 금액: 종목당 할당액 × 90%
  └─ 목적: 확신 (맞으면 집중 투자)
```

### **매도 로직 (우선순위):**
```
1순위: 손절 (-5%)
  └─ 빠른 정리 → 다른 기회 탐색

2순위: 타임컷 (30분 + 1% 미만)
  └─ 지루한 종목 교체

3순위: 트레일링 스탑 (TS)
  └─ 고점 추적 → 수익 극대화 ⭐

4순위: 목표 익절 (+10%)
  └─ 안전 장치
```

### **핵심 장점:**
```
✅ 리스크 최소화 (초기 10%)
✅ 승자 집중 (2차 90%)
✅ 빠른 손절 (-5%)
✅ TS로 수익 극대화
✅ 높은 승률
```

---

## 💧 **물타기 전략 상세**

### **매수 로직:**
```
1~10차 매수 (각 10%):
  ├─ 조건: -1% 하락 시마다
  ├─ 금액: 종목당 할당액 × 10%
  └─ 목적: 평단가 낮추기

매집 완료 후:
  └─ 손절 보호 해제 → -10% 도달 시 손절
```

### **매도 로직:**
```
1순위: 타임컷 (5분 + 0.3% 미만)
  └─ 빠른 교체 (물타기는 시간 싸움)

2순위: 트레일링 스탑 (TS)
  └─ +2.0% 활성화 → 수익 보호

3순위: 목표 익절 (+2.5%)
  └─ 낮은 목표 (빠른 회전)

4순위: 손절 (-10%)
  └─ 매집 완료 후에만 발동
```

### **핵심 장점:**
```
✅ 평단가 낮춤 효과
✅ 반등 시 빠른 익절
✅ 매집 중 손절 보호
```

### **주의사항:**
```
⚠️ 계속 하락 시 큰 손실
⚠️ 반등 없으면 물타기 실패
⚠️ 심리적 압박 큼
```

## 💻 **설정 방법**

### **1. UI에서 프리셋 선택:**
```
설정 탭 → 전략 프리셋 섹션
  ↓
원하는 버튼 클릭:
  - 🔥 몰빵 불타기
  - 💧 몰빵 물타기
  - 🔥 분산 불타기 (추천)
  - 💧 분산 물타기
  ↓
자동으로 최적 설정 적용
  ↓
'💾 설정 저장' 클릭
```

### **2. 수동 설정:**
```
매수 전략:
  - split_buy_cnt: 분할 횟수 (불타기: 2 / 물타기: 10)
  - initial_buy_ratio: 초기 매수 비율 (10%)
  - single_stock_strategy: 전략 (FIRE / WATER)
  - single_stock_rate: 추가 매수 간격 (불타기: 3% / 물타기: 1%)

매도 전략:
  - stop_loss_rate: 손절 (불타기: -5% / 물타기: -10%)
  - take_profit_rate: 익절 (불타기: 10% / 물타기: 2.5%)
  - trailing_stop: TS 활성화 (필수)
  - time_cut_minutes: 타임컷 시간 (불타기: 30분 / 물타기: 5분)
```

## ⚠️ **주의사항**

### **불타기:**
```
❌ +3% 도달 전 하락 시 → 즉시 손절 (-5%)
❌ 2차 매수 후 하락 시 → 큰 손실 가능
✅ 해결: 빠른 손절 + TS로 수익 보호
```

### **물타기:**
```
❌ 계속 하락 시 → 10회 매수 후에도 손실
❌ 반등 없으면 → 큰 손실 확정
✅ 해결: 종목 선정 신중 + 매집 완료 후 손절
```
